Admin Options

  • Diversinet OTP authentication page

  • Adding parameter '&oi=#' where # is the group number. This will change the logo shown at the top of the page to the group logo. It will also show only content associated with the selected group.
    This parameter needs only to be set ONCE for it to take effect. All subsequent page requests will use the value set.
    oi=0 is the default setting.

  • Adding parameter '&cd=1' will change delivery to open. The purchased content will be downloaded in the clear without any special headers or trigger files.
    The same conditions apply to the OI parameter.

Adding Content

Upload content to the site with the WEBDAV access path 'https://gruvi.sandisk.com/webdav/webdemo_data'.
All uploaded content will be added to the database when you view the Admin-Content page. The content entries must be updated with a valid TITLE field for it to be listed on the main content purchase page.
  • Content file = <content file name> (eg: superman.avi)
  • Content logo = <content file name>.jpg (eg: superman.avi.jpg)

Adding Groups

All groups currently need to be added to the database manually. There needs to be an associated logo image named operator#.jpg. # is the group number in the database.
